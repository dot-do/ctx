# Builder.Domains

**Domain platform** providing free `.do` subdomains and affordable domain registration for builders, developers, and creators.

## Overview

Builder.Domains removes the friction of getting a domain for your project. Get 10 free subdomains instantly, or register a custom domain with transparent pricing and instant provisioning.

**For Builders:**

## API Access

### Authentication

```typescript
import { DomainsClient } from '@dot-do/builder-domains'

const client = new DomainsClient({
  apiKey: process.env.BUILDER_DOMAINS_API_KEY
})
```

### Endpoints

**Subdomains:**
```typescript
// List subdomains
const subdomains = await client.subdomains.list()

// Create subdomain
const subdomain = await client.subdomains.create({
  name: 'myproject',
  target: 'myproject.pages.dev'
})

// Update DNS
await client.subdomains.updateDNS('myproject', {
  records: [
    { type: 'CNAME', name: '@', value: 'new-target.pages.dev' }
  ]
})

// Delete subdomain
await client.subdomains.delete('myproject')
```

**Domains:**
```typescript
// Search domains
const results = await client.domains.search('myproject')

// Register domain
const domain = await client.domains.register('myproject.com', {
  years: 2,
  autoRenew: true
})

// Manage DNS
await client.dns.setRecords('myproject.com', records)

// Transfer domain
await client.domains.transfer('myproject.com', {
  authCode: 'ABC123'
})
```
