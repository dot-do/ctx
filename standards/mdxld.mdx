# MDXLD Standard

**MDXLD** (MDX Linked Data) is a standard for embedding structured, machine-readable metadata into MDX documents using YAML-LD frontmatter.

## Overview

MDXLD combines:
- **YAML-LD** - YAML syntax for JSON-LD with `$` prefix
- **Markdown** - Human-readable content
- **Code** - JavaScript/TypeScript imports/exports
- **JSX** - React components for interactive UI

## Core Entity Types

1. **Function** - Executable code with inputs/outputs
2. **Component** - UI components (React, Vue, Svelte)
3. **Workflow** - Multi-step orchestrations
4. **Agent** - AI agents with tools and capabilities
5. **API** - RESTful endpoints
6. **Tool** - Software utilities
7. **Model** - AI/ML models
8. **Prompt** - AI prompt templates

## Quick Start

```bash

```

```mdx
---
$context: http://mdxld.org/context.jsonld
$type: Function
$id: https://example.com/functions/hello
title: Hello World
---

# Hello World Function

\`\`\`typescript
export function hello(name: string): string {
  return `Hello, ${name}!`
}
\`\`\`
```
