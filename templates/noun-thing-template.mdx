## Usage Examples

### Creating an Instance

```typescript
const entity = await db.things.create({
  ns: 'NAMESPACE',
  id: 'my-entity',
  type: 'Thing',
  data: {
    title: 'My Entity',
    description: 'Description here',
    property1: 'value',
    property2: 123
  }
})
```

### Querying

```typescript
// Get by ID
const entity = await db.things.find({
  ns: 'NAMESPACE',
  id: 'my-entity'
})

// Search by type
const entities = await db.things.findAll({
  type: 'Thing'
})

// Semantic search
const similar = await db.search('description of what you want', {
  type: 'Thing',
  limit: 10
})
```

### Updating

```typescript
await db.things.update({
  ns: 'NAMESPACE',
  id: 'my-entity'
}, {
  data: {
    property1: 'new value',
    updatedAt: new Date()
  }
})
```
