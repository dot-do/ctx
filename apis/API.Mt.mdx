## Core Features

### 1. MCP Server for All APIs

```typescript
import { APIMt } from '@api.mt/client'
import Anthropic from '@anthropic-ai/sdk'

const apimt = new APIMt({ apiKey: 'your-key' })
const claude = new Anthropic()

// Get MCP tools for specific categories
const tools = await apimt.mcp.getTools(['weather', 'email', 'maps'])

const response = await claude.messages.create({
  model: "claude-3-5-sonnet-20241022",
  tools: tools,
  messages: [{
    role: "user",
    content: "Find weather in user's location and send summary"
  }]
})
```

### 2. Semantic API Routing

AI determines best API for the task:

```typescript
// Natural language â†’ API route
const result = await apimt.route(
  "I need to send a text message",
  { context: { user, preferences } }
)

// Returns: Twilio SMS API (best match)
// Not: SendGrid (email), Slack (team messaging)
```

### 3. Auto-Generated LLM Tools

```typescript
// For OpenAI
const openaiTools = await apimt.generateTools('openai', ['payment', 'weather'])

// For Anthropic
const claudeTools = await apimt.generateTools('anthropic', ['maps', 'email'])

// For LangChain
const langchainTools = await apimt.generateTools('langchain', ['all'])
```

### 4. RAG Context from API Responses

```typescript
// Automatically embed API responses for context
const context = await apimt.executeWithRAG(
  'weather-api',
  { location: 'Seattle' },
  { embedResponse: true }
)

// Use context in next LLM call
const analysis = await gpt4.analyze(context)
```

## Use Cases

### AI Agent with Full API Access

```typescript
const agent = await apimt.createAgent({
  name: 'business-assistant',
  apis: ['calendar', 'email', 'crm', 'analytics'],
  llm: 'gpt-4'
```

