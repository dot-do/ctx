## Core Concept

### HATEOAS = Clickable APIs

Every API response includes navigation links, making APIs self-documenting and explorable:

```json
{
  "@context": "https://schema.org",
  "@type": "WeatherForecast",
  "@id": "https://api.ht/weather/forecast/san-francisco",
  "location": "San Francisco",
  "temperature": 72,
  "condition": "Sunny",
  "_links": {
    "self": {
      "href": "https://api.ht/weather/forecast/san-francisco"
    },
    "current": {
      "href": "https://api.ht/weather/current/san-francisco",
      "title": "Current weather"
    },
    "historical": {
      "href": "https://api.ht/weather/historical/san-francisco",
      "title": "Historical data"
    },
    "alerts": {
      "href": "https://api.ht/weather/alerts/san-francisco",
      "title": "Weather alerts"
    }
  }
}
```

**Click any link to explore** - no docs hunting required.
