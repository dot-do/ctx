# DB.HT

**Browse your database like the web.** No SQL required. Click through relationships, visualize schemas, and query data with the best DX in the industry.

## The Problem with Database APIs

**Traditional Approach:**
```sql
-- Step 1: Find customer
SELECT * FROM customers WHERE id = 123;

-- Step 2: Find their orders (copy customer_id manually)
SELECT * FROM orders WHERE customer_id = 123;

-- Step 3: Find order items (copy order_ids manually)
SELECT * FROM order_items WHERE order_id IN (456, 789);

-- Step 4: Join everything (write complex query)
SELECT customers.*, orders.*, order_items.*
FROM customers
JOIN orders ON customers.id = orders.customer_id
JOIN order_items ON orders.id = order_items.order_id
WHERE customers.id = 123;
```

**DB.HT Approach:**
```javascript
// Step 1: Get customer (includes navigation links)
GET /databases/shop/tables/customers/rows/123

// Response includes clickable links:
{
  "@type": "Customer",
  "id": 123,
  "name": "John Doe",
  "_links": {
    "orders": { "href": "/tables/orders?customer_id=123" }
  }
}

// Step 2: Click "orders" link
GET /tables/orders?customer_id=123

// Done! No SQL, no joins, just click.
```

```

  ↓ click "orders"

  ↓ click "items"
order_items?order_id=456
  ↓ click item "789"
order_items/789
  ↓ click "product"
products/999
```

### 3. Automatic Schema Introspection

Connect your database, we detect:
- All tables and columns
- Primary and foreign keys
- Relationships and constraints
- Indexes and types

**Then generate:**
- HATEOAS routes for all tables
- Navigation links for all relationships
- Visual schema diagram
- ER diagram

### 4. Live Data Preview

See your data update in real-time:
- WebSocket connections
- Change notifications
- Automatic refresh
- Optimistic updates

### 5. No SQL Required

Build complex queries visually:

**Query:** "Find all high-value customers who haven't ordered in 30 days"

**Visual Builder:**
```
1. customers table
2. JOIN orders (relationship)
3. FILTER: orders.total > $1000
4. FILTER: orders.date < 30 days ago
5. GROUP BY: customer
```

**Generated SQL:**
```sql
SELECT c.*, SUM(o.total) as lifetime_value
FROM customers c
LEFT JOIN orders o ON c.id = o.customer_id
WHERE o.created_at < NOW() - INTERVAL '30 days'
GROUP BY c.id
HAVING SUM(o.total) > 1000
```

But you never had to write SQL!

## Integration Example

### Connect Database

```typescript
import { db } from '@db.ht/client'

const database = await db.connect({
  name: 'shop',
  type: 'postgresql',
  host: 'db.example.com',
  database: 'production',
  user: 'readonly',
  password: '***'
})
```

### Navigate Data

```typescript
// Get customer (with HATEOAS links)
const customer = await database
  .table('customers')
  .get(123)

// Click "orders" link
const orders = await customer.follow('orders')

// Click first order's "items" link
const items = await orders[0].follow('items')

// All done with clicks, no SQL!
```

### Visual Query

```typescript
// Build query visually
const query = database
  .from('customers')
  .join('orders')
  .where('orders.total', '>', 1000)
  .groupBy('customer')

// Execute
const results = await query.execute()

// Results include HATEOAS links
results.forEach(row => {
  console.log(row._links) // Navigation links
})
```
