# API.net.AI

**Your API, generated by AI, deployed in seconds.**

Describe what you need in natural language, get production-ready REST APIs with OpenAPI specs automatically.

## How It Works

```typescript
// 1. Describe your API
const api = await apinetai.generate({
  description: `
    Create a task management API with:
    - Users can create, read, update, delete tasks
    - Tasks have title, description, due date, priority
    - Users can assign tasks to team members
    - Support filtering and sorting
  `
})

// 2. AI generates:
// - Database schema
// - REST endpoints (CRUD)
// - Authentication
// - OpenAPI spec
// - Deployment config

// 3. Auto-deploy
const url = await api.deploy()
// https://tasks-api-abc123.api.net.ai

// 4. Start using
const response = await fetch(`${url}/tasks`, {
  method: 'POST',
  headers: { 'Authorization': `Bearer ${apiKey}` },
  body: JSON.stringify({
    title: "Build feature",
    priority: "high"
  })
})
```
