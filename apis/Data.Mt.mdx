### 2. MCP Servers for Every Data Source

Direct integration with Claude and other LLMs:

```typescript
// Connect Claude to data sources via MCP
import Anthropic from "@anthropic-ai/sdk"

const anthropic = new Anthropic()

// Get MCP tools for specific data sources
const mcpTools = await data.mt.mcp.getTools([
  'census-demographics',
  'weather-forecasts',
  'company-data'
])

// Use with Claude
const response = await anthropic.messages.create({
  model: "claude-3-5-sonnet-20241022",
  max_tokens: 1024,
  tools: mcpTools,
  messages: [{
    role: "user",
    content: "Find the best cities for startups based on demographics and weather"
  }]
})

// Claude automatically queries Data.Mt sources!
```

### 3. Pre-Configured RAG Pipelines

Retrieval-Augmented Generation ready to go:

```typescript
// Create RAG pipeline from data sources
const rag = data.mt.createRAG({
  name: 'market-research',
  sources: [
    'census-demographics',
    'business-registry',
    'economic-indicators'
  ],
  strategy: 'mmr', // Maximal Marginal Relevance
  contextWindow: 10
})

// Retrieve context for LLM
const context = await rag.retrieve("What's the market opportunity for SaaS in Seattle?")

// Use with any LLM
const analysis = await openai.chat.completions.create({
  model: "gpt-4",
  messages: [
    { role: "system", content: `Context: ${context}` },
    { role: "user", content: "Analyze the market opportunity for SaaS in Seattle" }
  ]
})
```

### 4. Cross-Source Semantic Search

Search across multiple data sources simultaneously:

```typescript
// Search across census, weather, and business data
const results = await data.mt.crossSearch({
  query: "cities with growing tech sectors",
  sources: [
    { name: 'census-demographics', weight: 0.4 },
    { name: 'business-registry', weight: 0.4 },
    { name: 'economic-indicators', weight: 0.2 }
  ],
  filters: {
    population: { $gt: 500000 },
    techGrowth: { $gt: 10 } // 10% year-over-year
  }
})
```

### 5. Streaming Updates with Auto-Embedding

Real-time data with automatic re-embedding:

```typescript
// Subscribe to live updates
const stream = data.mt.stream('weather-forecasts', {
  location: 'Seattle',
  autoEmbed: true // Re-embed on updates
})

stream.on('update', async (data) => {
  // Data already has updated embedding
  console.log(data.temperature, data._embedding)

  // Immediately searchable semantically
})
```

## MCP Server Integration

### Connect Claude to Data Sources

```typescript
// Initialize MCP servers
const mcp = data.mt.mcp

// Get all available data source tools
const tools = await mcp.getAllTools()

// Or specific sources
const specificTools = await mcp.getTools([
  'census-demographics',
  'weather-forecasts'
])

// Use with Claude
const claude = new Anthropic()
const response = await claude.messages.create({
  model: "claude-3-5-sonnet-20241022",
  tools: specificTools,
  messages: [{
    role: "user",
    content: "Find population data for Seattle"
  }]
})

// Claude automatically calls Data.Mt MCP tools!
```
